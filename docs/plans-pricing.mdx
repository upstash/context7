## Plan Details

### Free Plan

The Free plan is perfect for individual developers who want to access public documentation.

**What's Included:**
- Access to all public repositories
- Basic access control
- 60 API requests per hour
- MCP server integration
- Community support via Discord and GitHub

**Limitations:**
- No private repository access
- No team collaboration
- Single user only

**Best For:**
- Individual developers
- Personal projects
- Learning and experimentation
- Public documentation needs

### Pro Plan

The Pro plan adds private repository access and team collaboration capabilities.

**Pricing:**
- **$7 per seat per month**: Each team member counts as one seat
- **$15 per 1M parse tokens**: For parsing private repositories (adding or refreshing)

**What's Included:**
- Everything in Free plan
- Private repository access
- Team collaboration (up to 20 members per project)
- 60 requests/hour per team member
- Email support
- Usage analytics and monitoring

**How Billing Works:**
1. **Member Seats**: Charged monthly based on the number of team members
2. **Parse Tokens**: Charged when adding or refreshing private repositories
3. **Pro-rated Charges**: Billed based on days in the current billing period
4. **Cached Pages**: Free when refreshing (only new/changed content is charged)

<Note>
Parse tokens measure how much *documentation* Context7 processes during ingestion—querying already-indexed docs stays free.
</Note>

**Example Costs:**
- 3 team members: $21/month (3 × $7)
- Adding a 500K token private repo: $7.50 one-time (0.5M × $15)
- Total first month: $28.50

**Best For:**
- Development teams
- Projects with private repositories
- Organizations needing collaboration
- Teams with proprietary documentation

## Pricing Details

### Parse Token Pricing

When you add or refresh a private repository, you're charged $15 per 1 million tokens parsed.

**What Counts as Parsing:**
- Initial addition of a private repository
- Refreshing a private repository (only new/changed content)
- Processing documentation and code examples

**What's Free:**
- Cached pages during refresh (unchanged content)
- Public repository access
- Querying documentation via API (covered by seat cost)

**Example Scenarios:**

<AccordionGroup>
  <Accordion title="Adding a Private Repository">
    - Repository documentation size: 750K tokens
    - Cost: $11.25 (0.75M × $15)
    - This is a one-time charge when adding the repository
  </Accordion>

  <Accordion title="Refreshing a Repository">
    - Previous size: 750K tokens
    - New content: 50K tokens
    - Cached (unchanged): 700K tokens
    - Cost: $0.75 (0.05M × $15)
    - You only pay for the new/changed content
  </Accordion>

  <Accordion title="Monthly Team Cost">
    - Team members: 5
    - Seat cost: $35/month (5 × $7)
    - Added 2 private repos: $30 (2M tokens × $15)
    - Refreshed repos: $5 (0.33M new tokens × $15)
    - Total: $70 for the month
  </Accordion>
</AccordionGroup>

### Rate Limits

| Plan | Rate Limit | Notes |
|------|------------|-------|
| Free | 60 req/hour | Per API key |
| Pro | 60 req/hour | Per team member |
| Enterprise | Custom | Based on your needs |

Rate limits are calculated using a sliding window algorithm and reset hourly.

## Frequently Asked Questions

### General Questions

<AccordionGroup>
  <Accordion title="What's included in the Free plan?" icon="gift">
    The Free plan includes access to public repositories, basic access control, and API usage with rate limits. It's perfect for individual developers who don't need private repositories or team collaboration.
  </Accordion>

  <Accordion title="How does the Pro plan pricing work?" icon="credit-card">
    Pro plan costs $7 per seat per month. Each team member counts as a seat, and you're billed based on the number of members in your team. You also pay for parse tokens used when adding or refreshing private repositories.
  </Accordion>

  <Accordion title="How much does parsing private libraries cost?" icon="dollar-sign">
    In addition to $7 per seat per month, you are charged $15 per 1M tokens for parsing private repositories. This charge applies when you add a new private repository or refresh an existing one. The cached pages (while refreshing private repositories) are free.
  </Accordion>

  <Accordion title="Can I access private repositories on the Free plan?" icon="lock">
    No, private repository access is only available on Pro and Enterprise plans. You'll need to upgrade to work with private repositories and collaborate with your team.
  </Accordion>
</AccordionGroup>

### Plan Management

<AccordionGroup>
  <Accordion title="How do I downgrade from Pro to Free?" icon="arrow-down">
    To downgrade, you must first remove all team projects and private repositories from your personal project. Once done, you can switch to the Free plan from the plans page.
  </Accordion>

  <Accordion title="Can I cancel my Pro subscription at any time?" icon="xmark">
    Yes, you can downgrade to the Free plan at any time. You will be charged a pro-rated amount for the days of your current billing period.
  </Accordion>

  <Accordion title="What's the difference between Pro and Enterprise?" icon="building">
    Enterprise plans include additional features like SOC-2 compliance, SSO (Single Sign-On), dedicated support, and custom rate limits. Contact us to discuss your specific enterprise needs.
  </Accordion>

  <Accordion title="How much does the Enterprise plan cost?" icon="envelope">
    Enterprise pricing is customized based on your specific needs and usage requirements. Please contact us at [context7@upstash.com](mailto:context7@upstash.com) to discuss pricing and features.
  </Accordion>
</AccordionGroup>

### Security & Privacy

<AccordionGroup>
  <Accordion title="How do you keep the privacy of my queries?" icon="shield">
    No user queries are sent to our servers. Instead, only generic topics are transmitted. Your actual prompts and code remain private on your local machine.
  </Accordion>

  <Accordion title="How do you ensure the code snippets are safe?" icon="check">
    We follow several processes to ensure safety:
    - Run an LLM-based model to detect injection attempts before storing code
    - Assign a trustScore to repository authors
    - Allow users to filter trusted libraries by organization and repository
    - Regular security audits and monitoring
  </Accordion>

  <Accordion title="What are your security practices?" icon="lock">
    We are part of Upstash Inc., a four-year-old Silicon Valley company. You can review our certifications and practices on our [Trust page](https://trust.upstash.com). We follow SOC-2 compliance standards and industry best practices.
  </Accordion>
</AccordionGroup>

### API Usage

<AccordionGroup>

If you need custom rate limits for high-volume usage, please contact us at [context7@upstash.com](mailto:context7@upstash.com) to discuss Enterprise plan options.

## Getting Started

### Free Plan

1. Sign up at [context7.com](https://context7.com)
2. Access your dashboard at [context7.com/dashboard](https://context7.com/dashboard)
3. Get your API key
4. Start using Context7 with public libraries

### Pro Plan

1. Start with a Free account
2. Go to [Plans & Pricing](https://context7.com/plans)
3. Click "Choose Pro Plan"
4. Add payment method via Stripe
5. Create team projects and add members
6. Add private repositories

### Enterprise Plan

1. Contact us at [context7@upstash.com](mailto:context7@upstash.com)
2. Discuss your requirements
3. Receive custom pricing proposal
4. Get onboarding support
5. Start using enterprise features
