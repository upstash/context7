Control which public libraries your team can access from the [libraries tab](https://context7.com/dashboard?tab=libraries).

## Public Library Access

![Public Library Access card](/images/dashboard/library-access-card.png)

Configure which public libraries your team can access:

![Library access options](/images/dashboard/library-access-selector.png)

**All Sources** (Default): Access to all public libraries, including future additions

**Select Specific**: Choose individual libraries or organizations

**Disable Public**: Private repos only (requires at least one private repo)

<Note>Only owners and admins can change these settings.</Note>

<Note>
  For private repositories, see the [Private Repositories](/dashboard/private-repositories) page.
</Note>
