Initial release of `@upstash/context7-ai-sdk` - Vercel AI SDK integration for Context7.

### Features

- **Tools**: `resolveLibrary()` and `getLibraryDocs()` tools compatible with AI SDK's `generateText` and `streamText`
- **Agent**: Pre-configured `Context7Agent` that handles the multi-step documentation retrieval workflow automatically
- **Prompts**: Exported `SYSTEM_PROMPT`, `AGENT_PROMPT`, `RESOLVE_LIBRARY_DESCRIPTION`, and `GET_LIBRARY_DOCS_DESCRIPTION` for customization
